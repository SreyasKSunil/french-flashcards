French Flashcards (no build, no Node)

What you get
- Runs in any browser. Works on iPhone and laptop.
- Import a CSV file with columns: fr,en,example,tags
- Tap card to flip
- Audio via device text-to-speech (fr-FR)
- Dark and light themes, both with gradients
- Saves progress in localStorage

How to run locally
- Open index.html in a browser
- On iPhone, host via GitHub Pages for best results

Host on GitHub Pages
1) Create a new repo, for example: flashcards-fr
2) Add index.html, styles.css, app.js, sample.csv
3) GitHub repo Settings -> Pages
4) Source: Deploy from a branch
5) Branch: main, folder: /root
6) Save. Your site link appears in Pages.

CSV format
Required headers: fr,en
Optional headers: example,tags
Use quotes if your text includes commas.
